Política de Privacidade — CantoMetro

Última atualização: 24 de abril de 2026

O CantoMetro é um aplicativo desenvolvido para competições de canto de pássaros. Esta política descreve como coletamos, usamos e protegemos suas informações.

1. Informações que coletamos

Coletamos as seguintes informações:

  • Nome e e-mail via login com Google
  • Dados de pássaros cadastrados (nome, espécie, idade, registro IBAMA)
  • Dados de medições (cantos registrados, timestamps, estatísticas)

2. Gravações de áudio

O aplicativo possui funcionalidade de gravação de áudio. As gravações são armazenadas exclusivamente no dispositivo do usuário e não são enviadas para nenhum servidor externo. O usuário é responsável por fazer backup de seus arquivos de áudio.

3. Como usamos suas informações

  • Para identificar o usuário e sincronizar seus dados entre dispositivos
  • Para armazenar o histórico de medições dos pássaros cadastrados
  • Para controle de planos de assinatura

4. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Google Firebase — para autenticação e armazenamento de dados
  • Google Play — para processamento de assinaturas

5. Segurança

Seus dados são armazenados com segurança no Google Firebase, protegidos por regras de acesso que garantem que apenas você pode visualizar e editar seus próprios dados.

6. Seus direitos

Você pode solicitar a exclusão de todos os seus dados a qualquer momento entrando em contato conosco pelo e-mail abaixo.
